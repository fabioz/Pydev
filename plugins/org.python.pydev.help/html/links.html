Where to find info about Pydev

  • To access the Pydev homepage, check: http://pydev.sf.net

    • This is the main page about Pydev


  • To access the Pydev Extensions homepage, check: http://www.fabioz.com/pydev

    • This page contains a 'getting started' for Pydev, info on getting commercial support and the Pydev Extensions plugin, which contains many enhancements to the Pydev 'Open Source' version


  • To access the blog that has information on its development, check: http://pydev.blogspot.com

  • To access Pydev at sourceforge, check: http://sourceforge.net/projects/pydev/

  • To access the Pydev bugtracker, check: Bug tracker

  • To access the Pydev feature requests, check: Feature Requests

  • If you have some doubt or want to talk to other Pydev users, check: Users Forum

  • If you want to code Pydev, check: Pydev code list
